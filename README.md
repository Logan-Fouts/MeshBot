# MeshBot
An off the grid meshtastic AI chat bot node.

## Overview
MeshBot is an AI-powered text only chat bot intended to operate off the grid using Meshtatic to provide potentially life saving information to users in isolated areas such as national parks. It will run on low power devices such as Raspberry Pi and communicate over long distances using LoRa technology.

### Details
- It operates on a network securred in a private Meshtastic channel.
- Users can chat with MeshBot after joining the Meshtastic network by scanning a QR code or entering a channel code.
- It uses a local instance of an AI language model to process and respond to user queries.
- It is designed to be energy efficient, allowing it to run for extended periods on battery power.
- It can provide information on a variety of topics including survival tips, first aid, navigation, and local flora and fauna.
