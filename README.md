# MeshBot

This is a super simple off-the-grid AI chat bot for Meshtastic, made for when you’re truly out of range.

## What is this?

MeshBot is a self-contained, text-only AI assistant that runs over Meshtastic’s LoRa network. It’s built for places with no cell service—think backcountry trails, remote camps, or national parks—where access to basic info can matter.

Once set up, join via QR code or channel key, and you can ask it for survival tips, first aid steps, basic navigation help, or details about local plants and wildlife. All without the internet.

## How it works

- **Runs locally** on low-power gear like a Raspberry Pi  
- **Talks over LoRa** using Meshtastic for long-range, low-bandwidth text  
- **Processes queries** with a local AI language model (no API calls, no cloud)  
- **Battery-friendly** so it can stay up for days off-grid and intended for solar power

## Use cases

- **Backpackers** wanting quick trail info or emergency guidance  
- **Preparedness groups** maintaining standalone comms with Q&A support  
- **Remote communities** with limited connectivity but need access to knowledge bases

## Getting started

1. **Set up** a Raspberry Pi (or similar) with Meshtastic and the MeshBot software  
2. **Power it** via battery or solar—configure for low-energy operation  
3. **Join the channel** by scanning the QR code it broadcasts or entering the channel key  
4. **Start chatting** by sending a text message over Meshtastic to the bot

## What you can ask

- *“How to treat a sprained ankle?”*  
- *“Edible plants in Pacific Northwest forests”*  
- *“How to signal for help in the mountains?”*  
- *“What wildlife is active at night here?”*  
- *“Basic steps for purifying water”*
